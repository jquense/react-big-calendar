# messages

String messages used throughout the component, override to provide localizations for button text, `title` attributes and more.

  time: 'Time',

  work_week: 'Work Week',

  month: 'Month',
  previous: 'Back',
  next: 'Next',
  yesterday: 'Yesterday',
  tomorrow: 'Tomorrow',
  today: 'Today',
  agenda: 'Agenda',

  noEventsInRange: 'There are no events in this range.',
